# AltaySec — Siber Güvenlik Mülakat Simülatörü

Siber güvenlik mülakatlarına hazırlanmak için geliştirilmiş interaktif soru-cevap platformu.

🔗 **Canlı Demo:** [GitHub Pages linkin buraya gelecek]

---

## Özellikler

- **150+ soru** — 9 farklı kategoride gerçek mülakat soruları
- **Çalışma Modu** — Kendi hızında ilerle, cevabı istediğinde gör, kendinle değerlendir
- **Simülasyon Modu** — Zorluk seviyesine göre geri sayım (60 / 90 / 120 saniye), süre dolunca otomatik değerlendirme
- **🔥 Sık Sorulan rozeti** — Gerçek mülakatlardan derlenen kritik sorulara otomatik etiket
- **Zorluk seviyeleri** — Her kategoriye göre Kolay / Orta / Zor
- **Hata Analizi** — Quiz sonunda yanlış cevaplanan sorular detaylı açıklamalarıyla listelenir
- **Kalıcı istatistikler** — Oturum geçmişi ve genel başarı oranı tarayıcıda saklanır
- **Klavye kısayolları** — `Space`, `G`, `Y`, `B`, `→`, `↓`

## Kategoriler
